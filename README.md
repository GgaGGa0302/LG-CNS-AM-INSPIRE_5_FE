## 👨‍👩‍👧‍👦 프로젝트 소개

가족 단위 방문객이 축제 정보 외에도 유모차 대여, 주차 가능 여부 등 편의 정보를 따로 찾아야 하는 불편함을 해결하기 위해 기획된 축제 정보 플랫폼입니다.

TourAPI를 통해 제공되는 기본 축제 정보에 더해, OpenAI가 생성하는 AI 요약을 결합하여 사용자 편의성을 높였습니다. 사용자는 축제를 검색하고, 상세 정보를 확인하며, '찜하기' 기능을 통해 자신만의 축제 목록을 관리할 수 있습니다.

## 🛠️ 기술 스택 (Frontend)

-   **Core**: React.js
-   **Styling**: Styled Components
-   **Routing**: React Router
-   **HTTP Client**: Axios
-   **State Management**: React Hooks (useState, useEffect)
